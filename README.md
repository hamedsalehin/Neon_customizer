# Neon Customizer ✨

A premium custom LED neon sign creator — interactive web app with SVG pathfinding, real-time glow rendering, and a liquid glass UI.

## Features

- 🎨 18+ neon-optimized fonts with real-time preview
- 🌈 12 neon colors including RGB Cycle and Color Flow animations
- 📐 SVG pathfinding engine for accurate strip length calculation
- 🖼️ 6 background environments + custom photo upload
- 📏 5 preset sizes + custom dimensions with auto-ratio
- 🔲 3 acrylic backboard styles (Cut to Shape, Cut to Letter, Rectangle)
- 💎 Liquid Glass UI aesthetic with backdrop blur effects
- 💰 Real-time pricing based on neon strip length

## Quick Start

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- **Frontend**: Vanilla HTML/CSS/JS with SVG rendering
- **Font Engine**: opentype.js for TTF path extraction
- **Server**: Express.js static file server
- **Design**: Liquid Glass aesthetic with CSS backdrop-filter
